## 字典基礎

### 什麼是字典？
字典是一種鍵值對(key-value pair)的資料結構，可想像成一本有索引的書，透過索引可以快速找到對應的內容。

### 字典的寫法
```python
# 空字典
empty_dict = {}

# 基本字典
player = {
    "name": "勇者",
    "hp": 100,
    "mp": 50
}
```

### 訪問字典元素
```python
print(f"角色名稱：{player['name']}")  # 使用中括號訪問
print(f"生命值：{player['hp']}")
```

## 字典的基本操作

### 修改數值
```python
player['hp'] = 150  # 更新生命值
```

### 新增項目
```python
player['level'] = 1  # 新增等級屬性
```

### 檢查鍵是否存在
```python
if 'skill' in player:
    print("已學會技能")
else:
    print("尚未學會技能")
```

## 字典的應用

### 管理角色屬性
```python
def show_status(char):
    print("\n=== 角色狀態 ===")
    for key in char:
        print(f"{key}: {char[key]}")

character = {

```
